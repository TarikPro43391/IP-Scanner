# IP-Scanner
IP Scanner v0.1.0
by TarikPro43391
Flask tabanlı, web arayüzlü IP çözümleme ve port tarama aracı. Domain/IP sorgulama, çoklu thread ile hızlı port tarama, servis tahmini ve banner grabbing özelliklerini tek panelde birleştirir. Sadece sahip olunan veya izin verilen sistemlerde test amaçlı kullanım için tasarlanmıştır.
Özellikler:

1•Domain → IP çözümleme + reverse DNS sorgusu
2•Çoklu thread port tarama (aralık veya liste girişi desteği)
3•Bilinen servis tahmini (HTTP, SSH, MySQL, Minecraft, RDP vb.)
4•Banner grabbing ile açık portlardan ek bilgi toplama
5•Canlı ilerleme çubuğu ile gerçek zamanlı tarama takibi
6•Windows için tek tıkla başlatma (baslat.bat)

Teknoloji: Python, Flask, HTML/CSS/JS
